# Sistema para avaliação de filmes

######Projeto construído na Semana Spring React 8.0
<br>

##### Nesse sistema é possível:
* Fazer a avaliação de um determinado filme.
* Validar campos...
<br>

##### Tecnologias usadas:
 * Spring Boot, JPA, H2, Security, lombok, Maven...
 * Angular e ReactJs
 * Git e GitHub
 * Postgres
 * Heroku e Netlify...
